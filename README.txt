# Wordpress-advanced-CPT
This repository is for grouping all custom php files, to work specific CPTs on specific wordpress themes.

First look 3 images of this CPT.

For this work, need 3 things:
1 - A site with wordpress and access to modify archive functions.php of your theme.
2 - ACF plugin (without galery) || ACF pro plugin (include galery).
3 - Any theme of yootheme or uikit-framework theme.
